Название сайта 
Мы создали сайт с помощью многих обучающих сайтов например как w3schools.com или же из видео с сайта youtube.com.
Для анимации мы использовали jQuery.
Для создания Админ панеля мы смотрели много обучяющих видео с youtube.com.
Наш сайт нужен для того чтобы смотреть новые фильмы.
Проблема было в том что у нас мало было времени на изучение некоторых функции java script.
